일반적인 브라우저의 웹페이지는 위와 같이 구성됩니다.

이를 참고하여 아래를 학습해주세요.

웹페이지의 구성

  • Head: 웹페이지의 아이콘(shortcut icon), 웹페이지의 이름(title) 등이 표기됩니다.
  • Body: 웹페이지의 다양한 콘텐츠가 표기됩니다.

위와 같은 구조를 통해 만들어지는 웹페이지는, 대표적인 3가지 언어를 통해 구체적으로 개발됩니다.

웹페이지의 개발언어

  • HTML: head 부분과 body 부분을 "태그" 를 통해 사이트 설정과 여러 요소 사용되는 언어
  • CSS: HTML 을 사용하여 만들어진 요소 에게 색상, 배치, 그림자 등 다양한 미적 효과나 구조를 구체적으로 꾸며주는 언어
  • JS: HTML과 CSS로 만들어진 정적인 요소들에게 "기능"을 주거나 요소들을 관리하는 프로그래밍 언어

위와 같은 3가지 언어는 서로 상호작용하며 다양한 웹페이지를 만들어갑니다.

저희 동아리는 HTML을 중심적으로 배우고, CSS의 기본적인 디자인법을 배우고, 이들에게 기능과 움직임을 주는 간단한 JS 문법을 배울 예정입니다.

[HTML]

HTML 이란?

"태그"를 통해 웹페이지의 body 속의 요소(예: 버튼, 이미지, 입력공간 등)를 만들거나,

head 부분의 title(웹페이지이름) 또는 shortcut icon(파비콘)을 설정하는 등 웹페이지의 "골격"을 만드는 언어

태그

태그들은 기본적으로 <태그명/> 또는 <태그명></태그명> 형태로 이루어져요.

태그마다 각자의 기능을 하고 (예시로 <img/> 태그는 이미지를 생성) 컴퓨터의 파일처럼 태그 속에 하위 태그가 들어가기도 합니다!

<태그명> 에서 태그의 이름 옆에 style, type, src 등 태그에 대한 다양한 설정이 가능합니다.

예시 : <img src="이미지이름" alt="이미지 설명" id="이 태그의 이름"/>

태그의 첫 번째 형태: 쌍태그

<태그명></태그명> 형태로 이루어지며, <태그명>과 </태그명> 사이에 다른 태그를 집어넣을 수 있어요.
    <태그1>
    <태그2/>
    </태그1>

위의 예시를 확인해볼까요?

태그1은 쌍태그이며, 하위 요소(또는 자식 요소)로 태그2를 포함하고 있습니다.

태그2에게 태그1은 "부모 요소"입니다.

중요한 점은 쌍태그 속에 또 다른 쌍태그를 넣을 수 있으며, 넣는 태그의 개수는 제한이 없습니다.

자유롭게 다른 태그를 추가할 수 있습니다.

태그의 두 번째 형태: 빈태그

<태그명/> 형태로 이루어지며, 하위 요소를 만들 필요 없이 자체적으로 기능을 가지는 경우가 대부분입니다.
    <태그1/>

빈태그는 일반적으로 위와 같이 사용됩니다!

CSS 이란?

style 태그 사이나 태그 옆에 style ="" 사이에 스며들어 html 을 아름답게 꾸며주는 언어입니다.

글자배치방향, 글자폰트, 글자크기, 글자색, 요소배경색, 요소배경 패턴, 요소 테두리, 테두리 디자인, 요소 공중에 띄우기, 요소 배치 방식 등등......이러한것들을 속성, 그 속성에 대한 값을 값 이라고 합니다

CSS 는 다양한 속성이 존재하며, 개발자가 모든 속성을 암기하기는 어렵습니다. 따라서 능동적으로 google 과 같은 플렛폼에서 자기에게 필요한 속성을 검색하는 자세가 필요합니다.

속성의 기본 사용구조와 대표적인 속성 몇가지를 알려드리겠습니다.

위에 html 부분에서 자세히 다루지는 않았지만...

HTML 태그에게는 속성이라는 개념이 존재합니다.

보통 아래와 같이 태그 이름 옆에 필요한 속성들을 추가합니다

    <태그명 속성1 = "속성내용" 속성2 = "속성내용" ....>

(앞에 img 태그의 src 나 input 태그의 type 도 속성의 예시입니다.

이제부터 혼동을 피하기 위해 html 속성 과 css 의 속성을 분리하여 표현하겠습니다

html 속성 중에서는 id, class 라는 속성들이 존재합니다.

  • id 는 대상 태그의 고유 이름이며 중복되지 않습니다.

  • class 는 비슷한 특징을 가진 태그들의 모임의 이름입니다.

예를 들어 class = "box" id = "1" 라면, 이 태그는 box 라는 반에 1 이라는 이름을 가진 태그인겁니다.

아이디와 클래스는 필수적인것은 아니고, js, css 에서 특정 태그를 지칭하기 위에 만들어져요.

    사용예시
    <태그명 id = "아이디이름" class = "클래스이름">

이렇게 클래스와 아이디를 사용하여 css 에서는 원하는 태그를 꾸미기 때문에, 이 개념은 중요하게 알고계셔야 합니다!

CSS 기본문법

기본적으로 css 속성과 그의 값은 아래와 같이 표기합니다.

    속성 : 값;

굳이 예시를 들자면

    background-color : red;

는 대상 태그의 배경색을 빨강으로 설정합니다.

참고로 쌍태그에 적용된 style 속성들은 그 태그의 자식태그에게 까지 적용됩니다.

그렇다면 대상태그를 어떤 방법으로 지칭할까요?

css 에서 html 을 꾸미는 방법은 두가지가 있습니다.

첫번째 방법 : style 태그 사용하기

아까 앞에서 알려드렸던 style 태그를 사용하여 html 요소들을 꾸며요

    <style>
    태그의 고유이름
    {
    속성1 : 값1;
    속성2 : 값2;
    }
    #아이디이름
    {
    속성1 : 값1;
    속성2 : 값2;
    }
    .클래스이름
    {
    속성1 : 값1;
    속성2 : 값2;
    }

예시를 들자면

    <div id = "box1" class = "boxClass">첫번째 div</div>
    <div id = "box2" class = "boxClass">두번째 div</div>
    <style>
    div
    {
    font-family : 궁서체;
    width : 100px;
    height : 100px;
    }
    #box1
    {
    background-color : blue;
    }
    #box2
    {
    background-color : green;
    }
    .boxClass
    {
    color : red;
    }
    </style>
    .boxClass:hover
    {
    color : yellow; /*마우스에 닿으면 노란색 글씨*/
    }
    </style>
를 적용하자면(이때 font-family 는 글씨체, background-color 는 배경색, color: 은 글자색 을 지정합니다.) 아래와 같이 됩니다. (#아이디,.클래스,고유태그이름 옆에 :hover 라는 표현을 추가하여, 그 대상 태그가 "마우스에 닿았는가" 상태가 되었을때 적용시킬 css 를 작성할수있습니다.)

두번째 방법 : html 태그의 속성 사용하기

두번째 방법은 비교적 간단합니다. 위에서 언급한 html 태그의 속성을 사용하는 것 입니다.

    <태그1 style = "css코드" (다른속성들...)>

이와 같은 방법을 사용하여 태그에게 id 나 class 를 부여하지 않고 css 코드를 적용시킬 수 있습니다.

이 상황에서도 마찬가지로 만일 "태그1" 이 자식요소를 가지고 있다면, style 속성 속의 css 속성들은 자식요소들에게도 적용됩니다.

예시를 들어볼까요?

아래는 자식요소로 img 를 가지고 있는 div입니다. img 태그에게는 src 를 지정해주지 않아서 이미지없음 에러가 생겨요.

    <div style = "width:100px; height:px; border:1px solid black;" id = "box">
    <img src = "" style = "background-color:red; width:50px;height:50px;" alt = "">
    </div>

css 속성을 html 에 적용시키는 법을 배웠으니, 이제 css 속성의 종류를 배워봅시다.

속성 이름 설명 예시
color 텍스트 색상을 설정합니다. color: red;
font-size 텍스트의 크기를 설정합니다. font-size: 16px;
background-color 요소의 배경색을 설정합니다. background-color: blue;
margin 요소의 바깥 여백을 설정합니다. margin: 10px;
padding 요소의 안쪽 여백을 설정합니다. padding: 10px;
border 요소의 테두리를 설정합니다. border: 1px solid black;
text-align 텍스트의 정렬을 설정합니다. text-align: center;
backdrop-filter: blur() 요소 뒤에 비춰지는 배경을 블러처리합니다. backdrop-filter: blur(2px);
opacity 투명도를 조절합니다. 값에는 0~1의 숫자가 들어갑니다. opacity:0.5;
width 요소의 너비를 설정합니다. width: 100px;
height 요소의 높이를 설정합니다. height: 100px;
display 요소의 표시 방식을 설정합니다. (예: block, inline, flex) display: flex;
position 요소의 위치를 설정합니다. (예: static, relative, absolute, fixed) position: absolute;
z-index 요소의 쌓이는 순서를 설정합니다. 숫자가 클수록 앞에 위치합니다. z-index: 10;
overflow 요소 내부의 콘텐츠가 넘칠 때 어떻게 표시될지를 설정합니다. (예: hidden, scroll, auto) overflow: hidden;
border-radius 요소의 모서리를 둥글게 만듭니다. border-radius: 10px;
box-shadow 요소에 그림자를 추가합니다. box-shadow: 2px 2px 5px grey;
transition CSS 속성의 변화를 부드럽게 전환시킵니다. transition: all 0.3s ease;
transform 요소를 변형시킵니다. (예: 회전, 크기 조절, 이동) transform: rotate(45deg);
flex-direction 플렉스 컨테이너 안에서 아이템들이 나열되는 방향을 설정합니다. (예: row, column) flex-direction: column;
align-items 플렉스 컨테이너 안에서 아이템들의 수직 정렬을 설정합니다. align-items: center;
justify-content 플렉스 컨테이너 안에서 아이템들의 수평 정렬을 설정합니다. justify-content: space-between;

실제로 css 의 속성은 이보다 훨씬 많습니다.

따라서 자신이 필요한 속성을 찾아보는 능력이 어느정도 필요합니다.

이제 이렇게 html 요소를 만들고, 이를 꾸미는 방법까지 배웠습니다.

하지만 이런 html 요소들에게 기능이 없으며, 상황에 따라 css 태그 적용도 변화시키고 싶을 수 있습니다.

따라서 이제 이 요소들에게 기능을 줄 수 있는 언어, js 를 배우겠습니다.

[JS]

JavaScript(JS) 란?

여기서 많은분들은 프로그래밍 언어라는 표현을 들어보셨을 것 같네요.

js 도 python 이나 php 등등의 언어처럼 프로그래밍 언어입니다.

js 는 node.js 나 다른 목적으로 사용되는 경우도 많지만, 저희는 프론트엔드, 즉 웹페이지의 사용자가 보게되는 화면에 대한 기능을 구현하기 위해 js 를 배우겠습니다.
